# Phishing Simulation & Social Engineering Awareness

## Overview

This project demonstrates a controlled phishing simulation conducted in a laboratory environment using GoPhish. The objective was to understand phishing attack techniques, user susceptibility, and the importance of security awareness training.

## Tools Used

- GoPhish
- Kali Linux
- Firefox Browser

## Project Objectives

- Create a simulated phishing campaign.
- Design a phishing email template.
- Create a mock login landing page.
- Configure a sending profile.
- Launch and monitor the campaign.
- Study phishing awareness concepts.

## Implementation

### User Group Creation
A sample employee group was created to act as recipients for the simulated phishing campaign.

### Email Template Design
An email template was created with the subject:

**Urgent: TourEnquiries Data Update Required**

The email requested users to review and update their information through a provided link.

### Landing Page Development
A demonstration login page was created to simulate a credential collection portal.

### Sending Profile Configuration
A mail profile was configured within GoPhish for campaign deployment and monitoring.

### Campaign Execution
The campaign was launched using the created user group, email template, landing page, and sending profile.

## Results

The GoPhish dashboard was used to monitor campaign activity and generate statistics.

Metrics observed:

- Emails Sent
- Emails Opened
- Links Clicked
- Submitted Data
- Emails Reported

## Security Awareness Recommendations

- Verify email senders before responding.
- Avoid opening suspicious links.
- Check website URLs before entering credentials.
- Report phishing attempts immediately.
- Use Multi-Factor Authentication (MFA).

## Skills Demonstrated

- Phishing Simulation
- Social Engineering Awareness
- Security Awareness Training
- GoPhish Configuration
- Cybersecurity Reporting

## Conclusion

This project provided practical exposure to phishing attack simulations and demonstrated the importance of user awareness in preventing credential theft and social engineering attacks.
